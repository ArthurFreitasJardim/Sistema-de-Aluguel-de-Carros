# 🚗 Sistema de Aluguel de Carros - Lab 02 (Sprint 1)

Este repositório contém a modelagem inicial e o desenvolvimento de um
sistema web para gestão de aluguéis de automóveis. O projeto segue a
arquitetura MVC e é desenvolvido em Java.
